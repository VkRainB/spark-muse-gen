## 关键决策
- 使用 Nuxt 4.3.0+（2025年7月稳定版发布）
- 使用 Nuxt UI 4 及语义化颜色（success, error, warning, info）
- 使用 Tailwind CSS（Nuxt UI 依赖，非 UnoCSS）
- 使用 Vite 7.x（Nuxt 4 内置）
- 使用 Pinia 配合 pinia-plugin-persistedstate
